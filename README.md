# GitHub Pull Request 練習リポジトリ（初級編）

ようこそ！🎉  
このリポジトリは、GitHubでPull Requestやブランチの操作を練習するための教材です。  
1人でも練習できるように、**コンフリクト発生**や**レビュー体験**の手順も含まれています。

---

## 🎯 目的
このリポジトリでできること：

- ブランチを作成する  
- Pull Request（PR）を作る  
- コンフリクトを解消する  
- レビューコメントを試す  

---

## 🪜 練習ステップ

### STEP 1. リポジトリをクローン
```bash
git clone https://github.com/<あなたのユーザー名>/github-pr-practice.git
cd github-pr-practice
